FAQ

How do I see this website?

Run cd website && python -m http.server 8080, then open http://localhost:8080/index.html.

How do I add Codunot to my server?

Use the Add to Server button on Home or Support pages.

How do I add Codunot as an app?

Use Add as App for DM/app install flow.

How do votes work?

Vote on top.gg to unlock advanced tools for 12 hours.

Where is support?

Support page links to owner/listing profile and official server.
