# 🤖 Codunot — AI That Lives Inside Discord

> Built by a 13-year-old developer.

Codunot brings powerful AI directly into your Discord server or DMs with lightning-fast replies (around **0.1–0.2s**).

Unlike tools like ChatGPT or Gemini, Codunot works **natively inside Discord** — no switching apps, no copy-pasting. Just AI built directly into your server.

Donations directly support development ❤️

## ❗ How To Use

### In servers (mention the bot)

```text
@Codunot your message
```

### In DMs

```text
Just type normally (no ping needed)
```

### Show the built-in help

```text
!codunot_help
```
